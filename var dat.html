variable JavaScript

  • are used to store information to be referenced & manipulated in a computer program.
  • In programming, a variable is a value that can be changed depending on the conditions or information being passed to the program.
  • Variables can be thought of as named containers. You can place data into these containers and then refer to the data simply by naming the container.
  • Before you use a variable in a JavaScript program, you must declare it. Variables are declared with the var keyword
  • You can also declare multiple variables with the same var keyword

Declaring JavaScript Variable

  • Var a;
  • Var name;
  • Var carName;

Assign value to the Variable

  • Var a =5;
  • Var name = “Angel”;
  • Var carName=“Aston”;

Data types

Data types

  • Data types are basically type of data that can be used and manipulated in a program.
  • For e.g. Numbers(1, 2,3), string(“hello World”),Boolean(true, false) etc.
  • To be able to operate on variables, it is important to know something about the type.
  • Without data types, a computer cannot safely solve.
  • When adding a number and a string, JavaScript will treat the number as a string.
  • JavaScript evaluates expressions from left to right.

Example

  • var x;           // Now x is undefined
    x = 5;           // Now x is a Number
    x = "John";      // Now x is a String
